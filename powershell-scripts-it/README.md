# 🖥️ PowerShell Scripts IT

Scripts PowerShell utiles pour l'administration système et réseau Windows.  
Développés dans le cadre d'une montée en compétences vers un poste d'administrateur réseau.

---

## 📁 Scripts disponibles

| Script | Description | Statut |
|--------|-------------|--------|
| `inventaire-systeme.ps1` | Inventaire RAM, CPU, disque, OS | 🔜 En cours |
| `sauvegarde-auto.ps1` | Sauvegarde dossier avec timestamp et log | 🔜 En cours |
| `creation-users-bulk.ps1` | Création d'utilisateurs locaux depuis un CSV | 🔜 En cours |

---

## ⚙️ Environnement testé

- Windows 10 / Windows 11
- PowerShell 5.1+
- Aucune dépendance externe requise

---

## 🚀 Utilisation

1. Clone le repo :
```powershell
git clone https://github.com/TON_USERNAME/powershell-scripts-it.git
```

2. Va dans le dossier :
```powershell
cd powershell-scripts-it/scripts
```

3. Lance un script :
```powershell
.\nom-du-script.ps1
```

> ⚠️ Si PowerShell bloque l'exécution, lance d'abord :
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```
